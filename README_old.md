# 华炎魔方低代码开发平台

华炎魔方是开源低代码开发工具，基于商业智能和模型驱动，可以按照业务需求，轻松便捷地创建智能化、移动化、个性化的应用程序。

华炎魔方致力于打造 Salesforce Platform 的开源替代方案。
